# **Agric-Empower - Combating The Soil Carbon Crisis**

A **regenerative agriculture platform** designed to address **The Soil Carbon Crisis** in the **Rhino Refugee Camp** community. This project tackles one of the most urgent environmental challenges of our time through **community-based soil restoration** and **climate-smart farming solutions**.

## üåç **The Environmental Challenge**

**The Soil Carbon Crisis** refers to the rapid loss of carbon stored in soils due to industrial agriculture, deforestation, poor land management, and climate change. As soil loses carbon, it becomes less fertile, releases CO‚ÇÇ into the atmosphere, accelerates global warming, and reduces global food security. **Agric-Empower** provides comprehensive regenerative solutions to reverse this critical environmental threat.

## üöÄ **Getting Started**

### **Prerequisites**

* **Modern web browser** (Chrome, Firefox, Safari, Edge)
* **Internet connection** for real-time data updates (optional)

### **Installation**

1. **Clone the repository**:

```bash
git clone https://github.com/yourusername/agric-empower.git
cd agric-empower
```

2. **Open the project in your browser**:

```bash
# Navigate to the frontend folder
cd frontend

# Open index.html in your browser
# Or use a local server:
python -m http.server 8000
# Then visit http://localhost:8000
```
